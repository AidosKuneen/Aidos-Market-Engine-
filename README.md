## 🧊 Overview

**Aidos Market** is a desktop cryptocurrency exchange demo designed to showcase a modern trading interface and simulated trading engine.

⚠️ This version is approximately **40% complete** and is intended for **demonstration purposes only**.

The focus of this release:

* Demonstrate the **core trading engine**
* Showcase the **UI/UX direction**
* Allow safe **simulated trading**

---

## 🪙 Supported Markets

* BTC / USDT
* BTC / USDC
* ETH / BTC
* ADK / BTC

## 💼 Features

* 🔐 Demo login system
* 💰 Starting balance: **$10,000 (simulated)**
* 📊 Candlestick-style chart (simulated data)
* 📈 Live price simulation
* 📉 Profit & Loss (PnL) tracking
* 🔄 Buy / Sell simulation
* 📘 Trade history
* 🌐 Simulated Aidos Network activity
* 🎨 Full **Aidos Market branding**
* 🧊 Splash screen on launch

---

## ⚠️ Important Notice

This is **NOT a live exchange**.

* No real funds
* No blockchain interaction
* No wallets connected
* All data is simulated locally

This demo is intended to showcase the **core trading engine and interface design only**.

## 🚀 Installation (Windows)

### Option 1 — Run Prebuilt EXE

Download and run:

```
Aidos Market Setup 1.0.0.exe
```

or

```
Aidos Market Portable 1.0.0.exe
```

---

### Option 2 — Run from Source

```bash
npm install
npm start
```

---

### Build EXE Manually

```bash
npm run dist
```

Output directory:

```
/dist-electron
```

## 🖥️ How to Use

1. Launch **Aidos Market**
2. Enter any username (demo login)
3. Access the trading interface

You can:

* Select trading pairs
* Simulate Buy / Sell orders
* Track portfolio value
* Monitor PnL
* Reset your account
